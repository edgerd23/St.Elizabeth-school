# 🏫 St. Elizabeth Nursery School — Official Website

A fully responsive, multi-page school website built for **St. Elizabeth Nursery School** — a trusted early childhood education institution nurturing young minds. This project delivers a clean, modern, and accessible web presence designed to inform parents, welcome new families, and streamline school-to-parent communication.

---

## 🌐 Overview

This website serves as the complete digital front for St. Elizabeth Nursery School, covering everything from program information and admissions to a fully functional parent portal. It was built using only **HTML5**, **CSS3**, and **Vanilla JavaScript** — no frameworks, no dependencies — keeping it lightweight, fast, and easy to maintain.

---

## 📄 Pages Included

- **Home** — Hero section with call-to-action buttons for enrollment and school information.
- **About Us** — School history, philosophy, core values, and a dedicated team section featuring staff profiles.
- **Programs** — Detailed breakdown of the Toddler (2–3yrs), Preschool (3–4yrs), and Pre-K (4–5yrs) programs with curriculum highlights.
- **Admissions** — Step-by-step enrollment process, tuition fees, sibling discounts, and a request information button.
- **Gallery** — Photo gallery showcasing school activities and moments from the school community.
- **Events & News** — Interactive events calendar with a news sidebar featuring school announcements and upcoming activities.
- **Contact** — Contact information, school hours, a message form, and a Google Maps placeholder for location display.
- **Parent Portal** — A secure, feature-rich parent dashboard with login authentication, progress reports, attendance calendar, messaging, photo uploads, and account settings.

---

## ✨ Key Features

- 📱 **Fully Responsive Design** — Optimized for desktop, tablet, and mobile with a hamburger menu for smaller screens.
- 🔐 **Parent Portal** — Secure login system with a full dashboard: children's profiles, progress reports, attendance tracking, event calendar with RSVP, teacher messaging, photo gallery, and profile/notification/security settings.
- 📅 **Events Calendar** — Visual monthly calendar with highlighted event dates and a live news sidebar.
- 📬 **Contact Form** — Functional contact form with client-side validation and submission feedback.
- 🎨 **Consistent UI** — Shared header, navigation, and footer across all pages for a seamless user experience.
- ⚡ **Vanilla JS** — No jQuery, no frameworks. Pure JavaScript for all interactivity including mobile menu toggling, smooth scrolling, tab navigation, form handling, and portal authentication.
- 🖼️ **School Branding** — Custom school logo integrated across all pages for brand consistency.
